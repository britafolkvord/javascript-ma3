1. In your own words, describe what JavaScript Code Style is.
-JavaScript is clean and easy to read. There are many methods that allow for more simplistic and readable code, which helps make complex code easy to understand.

2. What is a Regular Expression?
- Regular Expression are strings that represent a search pattern and they can help search, match and replace text.

3. What does the .random() method do?
- The random method produces a random number that can be anywhere between 0 and 1. 

4. What does the .sort() method do?
- The sort method in JavaScript sorts the items of an array. By default, it sorts the values as strings but it can also sort the values as numbers.

5. What does the .reverse() method do?
- The reverse method reverses the order of the items in an array, or objects containing a length property thus resembling an array.

6. List two examples of bad JavaScript code.
- The  double equal signs and global variables.

7. What data types can a JSON object contain?
- A JSON object can contain 6 data types: objects, arrays, strings, numbers, booleans and null.

8. What is minifying JavaScript Code?
- Minifying JavaScript code is the process of removing data from your code that is unnecessary or redundant without affaecting how the code is processed by the browser.


9. What is a global variable?
- A global variable is a variable declared outside a function, which makes it available to any other code in the document.
